# BrisJS-TypeScript

Demo app for BrisJS, showing TypeScript used in a life-like manner.

Prerequisites
-------------
Node.JS, and then:

```
npm install -g tsd gulp bower
```

Dependencies
----------
```
npm install
bower install
tsd install
```

Building
--------
`gulp`

or for watch support

`gulp watch`

Running
-------
After building, open `index.html` in the `dist/` folder.
